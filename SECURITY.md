# 安全与保密

本项目用于财务尽调工作流，不应包含真实客户资料。

## 禁止提交

- 真实审计报告、半年报、科目余额表、财务信息工作簿。
- 真实承租人、担保人、客户、供应商、银行、审计机构名称。
- 真实尽调报告成稿或中间版本。
- 合同编号、银行账号、税号、统一社会信用代码、联系人、电话、地址、身份证号。
- 盖章页、签字页、审计报告签章、银行流水、征信报告。

## 发布前必须检查

运行：

```powershell
.\scripts\check-sensitive-terms.ps1
```

同时人工检查所有 `.docx`、`.xlsx`、`.pdf`、`.md` 文件，确认不存在真实客户信息。

## 发现敏感信息

如果已经误提交敏感文件：

1. 立即将仓库设为 private。
2. 删除敏感文件。
3. 使用 Git 历史清理工具彻底移除敏感内容。
4. 轮换可能泄露的密钥、账号或访问凭据。
5. 按所在机构的数据安全制度上报。
