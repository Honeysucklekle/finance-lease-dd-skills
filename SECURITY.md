# 安全与保密

本仓库聚焦财务尽调工作流与说明文档，默认不收录真实业务资料。

## 建议排除在仓库之外的内容

- 真实审计报告、半年报、科目余额表、财务信息工作簿。
- 真实承租人、担保人、客户、供应商、银行、审计机构名称。
- 真实尽调报告成稿或中间版本。
- 合同编号、银行账号、税号、统一社会信用代码、联系人、电话、地址、身份证号。
- 盖章页、签字页、审计报告签章、银行流水、征信报告。

## 建议的仓库检查

运行：

```powershell
.\scripts\check-sensitive-terms.ps1
```

同时人工检查所有 `.docx`、`.xlsx`、`.pdf`、`.md` 文件，确认样例内容已完成脱敏或替换为合成数据。

## 如误纳入敏感信息

如果敏感文件或字段已经进入仓库历史：

1. 立即将仓库设为 private。
2. 删除敏感文件。
3. 使用 Git 历史清理工具彻底移除敏感内容。
4. 轮换可能泄露的密钥、账号或访问凭据。
5. 按所在机构的数据安全制度上报。
