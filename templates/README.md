# 模板目录

本目录用于放置脱敏后的模板文件。

建议文件：

- `financial-statement-template.xlsx`：空白财务信息工作簿模板。
- `due-diligence-template-sanitized.docx`：脱敏后的尽调 Word 模板。

## 注意

不要直接放入真实项目中的样式参考 Word，除非已经完成彻底脱敏。

发布前请检查：

- 文档属性。
- 修订记录。
- 批注。
- 隐藏文本。
- 隐藏 sheet。
- 外部链接。
- 公司名称、客户名称、供应商名称、金额等真实信息。

第一版建议只保留本 README，不上传任何 Office 文件。确认模板完全脱敏后再添加。
