# 模板目录

本目录用于放置脱敏后的模板文件或模板占位说明。

建议文件：

- `financial-statement-template.xlsx`：空白财务信息工作簿模板。
- `due-diligence-template-sanitized.docx`：脱敏后的尽调 Word 模板。

## 使用建议

不建议直接放入真实项目中的样式参考 Word，除非已经完成彻底脱敏。

公开分享前建议检查：

- 文档属性。
- 修订记录。
- 批注。
- 隐藏文本。
- 隐藏 sheet。
- 外部链接。
- 公司名称、客户名称、供应商名称、金额等潜在识别信息。

若模板尚未完成充分脱敏，可仅保留本 README，待模板清理完成后再补充 Office 文件。
