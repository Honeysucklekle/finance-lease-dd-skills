# 数据材料要求

## 必需材料

- 三年审计报告。
- 最新一期科目余额表或科目明细表。
- 用户整理的财务信息 Excel。
- 尽调模板或样式参考 Word。

## 财务信息 Excel 建议包含

- 基本信息。
- 资产负债表。
- 利润表。
- 现金流量表。
- 财务指标。
- 重要科目。

若 Excel 已经包含万元口径表，优先使用万元口径，不要重复从元口径换算。

## 科目余额表建议包含

- 科目编码。
- 科目名称。
- 核算维度名称。
- 期初余额。
- 本期借方发生额。
- 本期贷方发生额。
- 期末余额借方。
- 期末余额贷方。

## 审计报告重点读取

- 财务报表主表。
- 应收款项账龄和坏账准备。
- 存货分类和跌价准备。
- 固定资产分类、折旧和减值。
- 借款分类、担保方式和抵押质押。
- 长期应付款、租赁负债、售后回租。
- 股本、资本公积、未分配利润。
- 受限资产、担保、或有事项。

## 缺失材料处理

- 没有最新一期科目余额表时，使用半年报或管理报表附注。
- 没有最新一期前五大时，才使用上一年审计附注前五大。
- 没有审计报告附注明细时，可使用科目余额表拆分，但要说明口径。
