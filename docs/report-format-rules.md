# 正式尽调格式规则

## 主要科目情况分析

每个科目按以下顺序：

```text
科目标题
科目明细表
文字说明
前五大表格，如有
```

有前五大的科目不得写成：

```text
科目标题
科目明细表
前五大表格
文字说明
```

## 前五大披露

正式尽调只披露最新一期前五大。

表头建议为：

```text
单位名称 | 最新期末余额 | 占比
```

不要保留：

```text
2024单位名称 | 2024年末余额 | 2024占比 | 2025单位名称 | 2025年末余额 | 2025占比
```

只有最新一期没有前五大资料时，才披露上一年审计附注前五大，并说明来源。

## 不写入正式报告的底稿内容

- 入选科目及占比汇总。
- 独立的报表调整/重分类口径说明。
- 只用于底稿勾稽的说明大表。
- 双期间前五大对照表。

## 字体和表格

以样式参考 Word 为准。

常见要求：

- 普通正文宋体，小四或样式指定字号。
- 正文首行缩进。
- 行距与样式参考一致。
- 表格边框、列宽、行高和数字字体与样式参考一致。
- 不要误加粗普通科目标题、表格正文或分析正文。

## 现金流量表

现金流量表标题行数值列不得填 0，例如：

- 一、经营活动产生的现金流量：
- 二、投资活动产生的现金流量：
- 三、筹资活动产生的现金流量：

这些行是标题行，不是金额行。
