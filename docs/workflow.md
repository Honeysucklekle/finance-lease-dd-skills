# 工作流程

## 1. 准备输入材料

典型输入通常包括：

- 最近三年审计报告。
- 最新一期科目余额表或科目明细表。
- 若没有科目余额表，可补充最新一期半年报或管理报表。
- 结构化财务信息工作簿，至少包含三年一期报表、财务指标和 `重要科目` sheet。
- 尽调模板与样式参考 Word。

## 2. 生成或复核重点科目

通常先使用 `financial-key-subjects`：

- 以资产负债表列示金额为控制总数。
- 资产类科目按总资产 `5%` 筛选。
- 负债及权益类科目按负债与权益总额 `3%` 筛选。
- 业务上要求补充披露的往来科目，即使未达阈值，也可纳入披露清单。
- 生成或复核 `重要科目` sheet。

## 3. 复制模板并保留样式

推荐在复制件上完成写作和排版，不直接修改源模板或原始材料。生成新 Word 文件时，通常优先保留：

- 字体和字号。
- 行距和缩进。
- 表格列宽、边框、底纹。
- 标题层级和编号样式。

## 4. 组织正式报告

使用 `financial-lessee-due-diligence` 时，通常需要：

- 写入三年一期报表和财务指标。
- 依据 `重要科目` 清单组织主要科目分析。
- 结合融资租赁场景，围绕资产质量、负债期限结构、经营现金流和偿债能力形成分析。
- 对应收、其他应收、应付、其他应付等存在前五大明细的科目，按规则披露最新一期前五大。

## 5. 正文与底稿的取舍

正式报告通常不直接搬入底稿全部内容。

一般不写入正式报告的内容包括：

- 入选科目及占比汇总。
- 独立的报表调整/重分类口径说明板块。
- 双期间前五大底稿表。

通常保留在正式报告中的内容包括：

- 科目明细表。
- 文字分析。
- 最新一期前五大表格。

## 6. 交付前复核

交付前建议至少检查：

- Word 文件能正常打开。
- 源文件未被修改。
- 主要科目无遗漏。
- 有前五大的科目顺序为：科目明细表、文字说明、前五大表格。
- 前五大仅披露最新一期。
- 表格合计与财务报表列示金额一致。
- 文字金额与表格金额一致。
- 无样式参考残留信息或占位符。
