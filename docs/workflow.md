# 工作流程

## 1. 准备材料

每次撰写承租人财务尽调前，准备以下材料：

- 最近三年审计报告。
- 最新一期科目余额表或科目明细表。
- 若没有科目余额表，则提供最新一期半年报或管理报表。
- 用户整理的财务信息工作簿，至少包含三年一期报表、财务指标、重要科目。
- 尽调模板和真实样式参考 Word。

## 2. 生成或复核重要科目

先使用 `financial-key-subjects`：

- 以资产负债表为控制总数。
- 资产类科目按总资产 5% 筛选。
- 负债及权益类科目按负债与权益总额 3% 筛选。
- 用户指定披露的往来科目即使未达阈值，也作为补充披露。
- 生成或复核 `重要科目` sheet。

## 3. 复制模板

不得直接修改模板或源文件。应复制模板或样式参考，生成新的尽调 Word 文件。

优先保留：

- 字体和字号。
- 行距和缩进。
- 表格列宽、边框、底纹。
- 标题层级和编号样式。

## 4. 撰写正式尽调

使用 `financial-lessee-due-diligence`：

- 把三年一期报表和财务指标写入报告。
- 根据 `重要科目` 清单写主要科目情况分析。
- 对每个重点科目写明余额、变化、构成、原因和风险。
- 对应收、其他应收、应付、其他应付等有前五大的科目，披露最新一期前五大。

## 5. 正式报告取舍

正式报告不直接搬入底稿全部内容。

正式报告不写：

- 入选科目及占比汇总。
- 独立的报表调整/重分类口径说明板块。
- 双期间前五大底稿表。

正式报告应写：

- 科目明细表。
- 文字分析。
- 最新一期前五大表格。

## 6. 交付前复核

至少复核：

- Word 能正常打开。
- 源文件未被修改。
- 主要科目无遗漏。
- 有前五大的科目顺序为：科目明细表、文字说明、前五大表格。
- 前五大只披露最新一期。
- 表格合计与财务报表列示金额一致。
- 文字金额与表格金额一致。
- 无真实样式参考公司残留。
- 无占位符和 Markdown 标记残留。
