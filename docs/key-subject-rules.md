# 重点科目规则

## 资产类科目

资产类科目按总资产 5% 筛选。

```text
资产科目占比 = 该资产科目账面价值 / 资产总计
```

最近审计年度期末或最新一期期末任一期间超过 5%，即入选。

## 负债及权益类科目

负债及权益类科目按负债与权益总额 3% 筛选。

```text
负债/权益科目占比 = 该科目账面价值 / 负债和所有者权益总计
```

最近审计年度期末或最新一期期末任一期间超过 3%，即入选。

## 补充披露

业务上明确要求披露的科目，即使不达阈值，也可作为补充披露。

常见补充披露：

- 其他应收款。
- 其他应付款。
- 应收账款前五大。
- 应付账款前五大。
- 预付款项。
- 合同负债。

## 不应作为重点科目的项目

- 流动资产合计。
- 非流动资产合计。
- 资产总计。
- 流动负债合计。
- 非流动负债合计。
- 负债合计。
- 所有者权益合计。
- 负债和所有者权益总计。
- 财务指标表中的派生指标。

## 勾稽要求

每个重点科目明细表都应勾稽到资产负债表列示金额。

若明细合计与报表列示不一致，可列示：

```text
报表调整/重分类
```

该行不是实际二级科目，也不是往来单位，而是用于衔接余额表口径与报表口径的说明项。
